Greatest common factor calculator finds the GCF of a set of numbers and all factors of these numbers. Solutions with steps for finding the GCF are also demonstrated.

[https://www.calculator.io/gcf-calculator/](https://www.calculator.io/gcf-calculator/)

== Usage ==

Add the GCF Calculator shortcode to your page, post or sidebar:

`[ci_gcf_calculator]`

Add the following code to your template where you would like the GCF Calculator to appear:

`<?php display_ci_gcf_calculator(); ?>`

== Installation ==

1. Upload the GCF Calculator /ci_gcf_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the GCF Calculator plugin through the "Plugins" menu in WordPress.
